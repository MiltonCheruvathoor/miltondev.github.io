# MiltonDev GitHub Pages

This repository hosts the public website for **MiltonDev** projects.  
It provides a central place for project information, documentation, and privacy policies.

## Current Content
- **Homepage**: [https://miltondev.github.io](https://miltondev.github.io)  
- **Privacy Policy**: [https://miltondev.github.io/privacy.html](https://miltondev.github.io/privacy.html)

## Purpose
This site is maintained to:
- Provide official privacy policies for apps such as **Audio Concatenator Pro**.
- Offer a clean landing page for MiltonDev projects.
- Serve as a central hub for future project documentation.

## Notes
- All files in this repository are static HTML/CSS and are deployed via **GitHub Pages**.
- No personal data is collected or stored by MiltonDev applications.
